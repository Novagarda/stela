Este es un post de ejemplo escrito en **Markdown**. Eleventy lo convierte a HTML y lo envuelve con el layout de blog.

## Qué puedes hacer

- Escribir contenido en `.md` dentro de `src/blog/posts/`
- Usar front matter para título, fecha y descripción
- Añadir imágenes en `src/assets/` y referenciarlas con rutas absolutas

```js
// También puedes incluir bloques de código
console.log("Hola desde Stela");
```

Cada archivo nuevo en `posts/` aparecerá automáticamente en el índice del blog.
